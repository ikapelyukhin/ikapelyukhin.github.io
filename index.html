Tools used

  • MicMac – for generating 3D point clouds from photos. Has pretty good tutorials;
  • Meshlab – for generating meshes from point clouds. Compiling it was a bit of a pain;
  • Blender – for rotating/centering the resulting meshes and exporting to Three.js JSON format;
  • Three.js – for rendering the resulting models in the browser.
